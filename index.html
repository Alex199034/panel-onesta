Cargar nuevo Panel de Control
Subí un nuevo archivo Excel para actualizar todos los datos sin tener que mandármelo. El procesamiento se hace en el navegador.
📊
Seleccioná el archivo Panel de Control
El archivo debe ser el Excel del Panel de Control (.xlsx) con las columnas Fecha, Local, Dirección, Nombre, Entrada planificada, Salida planificada.
✔ Archivo procesado correctamente
💡 También podés usar un servidor local:
1. Abrí PowerShell en la carpeta del archivo
2. Ejecutá: python -m http.server 8080
3. Abrí en Chrome: http://localhost:8080/panel_control_rutinas.html
4. Esto también activa la geocodificación automática
